Cluckles


Cluckles is a live theme editor that leverages Less to Customise Twitter Bootstrap (In the future I will look into adding support for other CSS Frameworks).

Cluckles is changing, it will be moving to be a Node-Webkit based app. If you have any questions/thought about this please feel free to email me tom@ilikeprograms.com

Create a Theme! Download Source (zip) Also on Bower
Current release v1.0.3


Features of Cluckles


Lets you Fully Customise Bootstrap v3.3.3

You can add Custom Less and CSS

Import/Export support so you can resume later

Doesnt cost a thing! Free and Open Source (MIT)

You can name your themes and add extra metadata such as Author/Email etc

Can even import Variables.less files, so you dont have to enter them in!

Can help quickly prototype new Themes, so you can leave early and get a Coffee!

You can use the Editor or Download the source and Embed the editor in another project

Has Undo/Redo inbuilt, so you dont have to pull your hair out
